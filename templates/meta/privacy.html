Privacy Policy

The information collected is as follows:

  • Web server access logs (Source IP address, browser user-agent, page requests, etc).
  • Email address and salted/hashed password.
  • Application events and metrics (function execution and the user who triggered).

Logs are checked and events are captured for troubleshooting purposes only. None of this data is shared with any third-party.
